C语言规定了一些单词不允许用作标识符，这些单词称为关键字（Keyword）或保留字 （Reserved Word)。
# list of reserved words 
1. auto 
2. break 
3. case 
4. char 
5. const 
6. continue 
7. default 
8. do 
9. double 
10. else 
11. enum 
12. extern 
13. float 
14. for 
15. goto 
16. if 
17. inline 
18. int 
19. long 
20. register 
21. restrict 
22. return 
23. short 
24. signed 
25. sizeof 
26. static 
27. struct 
28. switch 
29. typedef 
30. union 
31. unsigned 
32. void 
33. volatile 
34. while 
35. _Bool 
36. _Complex 
37. _Imaginary
