用于定义了变量之后，把值存到它们的存储空间里，形式为:
```C
a = b + c;
```
等号左边表示存储位置，称为左值 （lvalue）。
等号右边表示要存储的值，可以是任意组合的表达式（[[expression|statement]])，所以通常所说的表达式的值也称为右值（rvalue）。
如果所赋的值和变量的类型不符会导致编译器报警或报错
