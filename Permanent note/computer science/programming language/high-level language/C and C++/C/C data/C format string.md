用于printf中进行格式化输入输出。它规定了后面的几个数据以何种格式插入到字符串中。它们只在格式化字符串中占位，并不出现在最终结果中。所以它们也被称作占位符（placeholder）。 
```C
%d  //整型
%c  //字符型
%s  //字符串型
%f  //浮点型
```
