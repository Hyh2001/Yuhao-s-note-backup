# C语言程序基本要求：
1. 程序总是从main的第一条语句开始执行。
2. 每条语句末尾有一个“;”号。
3. C语言使用“{  }”大括号（brace）来把语法结构组织成块。 
4. 块中的语句要有适当缩进（indent）

# C语言程序的基本结构
```C
#include 

int main(){
	a = b + 1;

}
```
