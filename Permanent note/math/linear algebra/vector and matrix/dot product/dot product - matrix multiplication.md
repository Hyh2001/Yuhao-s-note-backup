In matrix multiplications, dot product can be implemented between a [[row vector]] and a [[column vector]]. This two vectors must have the same dimensions. $a^{row} \cdot b^{col} = a_1b_1 + a_2b_2 + a_3b_3 + ... + a_kb_k$
