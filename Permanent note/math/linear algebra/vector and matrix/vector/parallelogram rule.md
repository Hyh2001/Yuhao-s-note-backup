Parallelogram rule is used to add two vectors.

# properties of parallelogram rule
1. commutative
    $\vec{r} + \vec{s} = \vec{s} + \vec{r}$
2. associative
    $(\vec{r} + \vec{s}) + \vec{t} = \vec{r} + (\vec{s} + \vec{t})$
3. distributive
    $(a+b)\vec{r} = a\vec{r} + b\vec{r}$
4. additive inverse
