#calculus 
This inequality is in the form of $|a+b| \leq |a| + |b|$. We can proof this inequality by using [[Cauchy-Schwarz inequality]]: 
$||U + V||^{2}= ||U||^{2}+||V||^{2}+2U \cdot V \leq ||U||^{2} + ||V||^{2} + 2||U||||V|| = (||U|| + ||V||)^2$  
Therefore: 
$||U+V||\leq||U||+||V||$ 
# applications
1. It is used to quickly estimate the accuracy of a sum of approximations
2. It is used to relate distances between numbers on the number line by: 
$|z - x| = |(z-y) + (y-z)| \leq |z-y| +|y-x|$
