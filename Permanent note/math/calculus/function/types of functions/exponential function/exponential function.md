#calculus/function #review #拆分

It is a function of form $f(x) = b^x$

## calculation laws

1. $b^{x+y} = b^x b^y$
2. $b^{x-y} = \frac{b^x}{b^y}$
3. $(b^x)^y = b^{xy}$
4. $(ab)^x =  a^x b^x$
