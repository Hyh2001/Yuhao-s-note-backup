# well ordered set

## well ordered set

一个集合的任意非空子集都有一个最小元素

### 推广1

对任意非负整数$n$，大于等于$-n$的整数构成的集合是良序的

#### proof

1.$S$表示$\geq -n$的整数构成的集合
2.$S+n$得到非负整数集，符合良序原理，任意非空子集都有最小元素$m$
3.可知对于$S$,任意非空子集都有最小元素$m-n$

### 推广2

定义实数集合S的下界或上界b，满足对任意$s  \in S$都有$b \leq s$ 或者$b \geq s$

#### 推论1

任何有下界的整数集合都是良序的

#### 推论2

任何有上界的非空整数集合都存在1个最大的元素
